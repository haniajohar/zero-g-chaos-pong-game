# 🚀 Zero-G Chaos Pong

Welcome to **Zero-G Chaos Pong**—a high-velocity, physics-bending web arcade game where gravity is optional and chaos is guaranteed. Built entirely using web technologies, you can play this right in your browser.

---

## 🎮 Game Overview
Traditional Pong meets unpredictable space physics. In a zero-gravity arena, the ball doesn't just bounce—it accelerates, curves, and reacts to dynamic anomalies, making every round unpredictable.

*   **The Concept:** Classic 2D arcade Pong, but with drifting momentum, shifting gravity wells, and chaotic modifiers.
*   **Tech Stack:** Vanilla HTML5 Canvas & JavaScript (No heavy engines, purely lightweight browser performance).
*   **Target Platforms:** Modern Web Browsers (Desktop/Laptop).

---
